# majordomo-mcp
MCP-MajorDoMo-Xiaozhi


🏠 MCP Bridge для MajorDoMo — Умный дом с ИИ

Лицензия Python Платформа


MCP Bridge — это полноценная система управления умным домом на базе MajorDoMo, интегрированная с ИИ-агентом xiaozhi через протокол MCP (Model Context Protocol).

Поддерживает управление голосом, веб-интерфейс, расписание и сценарии.


📌 Обзор

Этот проект обеспечивает полный мост между MajorDoMo (открытой системой автоматизации дома) и ИИ-агентом с использованием протокола MCP. Включает:

✅ MCP-сервер — подключает ИИ к MajorDoMo

✅ Веб-панель — создание и редактирование устройств, настройка

✅ Планировщик — автоматизация по расписанию

✅ Работает на Raspberry Pi, Linux-серверах.

🚀 Быстрый старт

Требования

Сервер Linux или Raspberry Pi

Python 3.10+


Голосовые команды (через xiaozhi)

«Включи свет в комнате отдыха»

«Выключи всё»

«Какая температура в парной?»
