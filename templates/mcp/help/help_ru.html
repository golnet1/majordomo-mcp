Общие сведения

Модуль XiaoZhi MCP предназначен для поддержки навыков для платформы XiaoZhi AI. Модуль реализует авторизационный сервис на основе стандарта MCP промежуточный API, который принимает на вход запросы в формате платформы MCP и преобразует их в запросы к API MajorDoMo.

Для успешного функционирования модуля и управления устройствами, подключенными к MajorDoMo, через XiaoZhi AI требуется, в первую очередь, зарегестрироваться в сети Интернет и обеспечить тем самым их доступность со стороны облака XiaoZhi AI. Во вторую очередь, необходимо в консоли разработчика платформы XiaoZhi AI войти в раздел Configure Role -> MCP Settings -> Get MCP Endpoint и скопировать Endpoint URL внести в настройки модуля MCP в разделе Настройки пункт Endpoint URL.

В модуле MCP все обработчики запросов реализованы с помощью Model Context Protocol (MCP) – универсальный протокол для взаимодействия с ИИ.

В конечном итоге запуск модуля сводится к такой последовательности шагов:

  1. Установка модуля из маркета дополнений.
  2. Регистрация в XiaoZhi AI.
  3. Настройка модуля.
  4. Добавление и управление устройствами в модуле.
